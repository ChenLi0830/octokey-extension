# Octokey-Extension
The extension of Octokey. A user needs it to make full use of the Octokey web app. 

It grants the web app browser permissions such as 
* making cross-origin request
* allowing communication between tabs
* using local storage

Targeted Browsers:
- [x] Chrome
- [ ] Safari
- [x] Baidu Browser
- [x] 360 Browser
- [x] CM(Liebao) Browser
- [ ] Sogou Browser
